3. 핵심 데이터 모델

Database Schema (Supabase PostgreSQL)
👤 users
id (uuid, PK)
toss_user_id (unique)
nickname (varchar)
hourly_wage (integer)
poop_coins (integer, default 0)
streak_days (integer, default 0)
created_at (timestamptz)
✅ check_ins
id (uuid, PK)
user_id (FK → users)
started_at (timestamptz)
ended_at (timestamptz, nullable)
duration_seconds (integer)
earned_amount (integer)
coins_earned (integer)
mode ('quick' | 'timer')
🎁 rewards
id (uuid, PK)
user_id (FK → users)
type ('ad' | 'streak' | 'achievement')
coins_amount (integer)
toss_points (integer, nullable)
created_at (timestamptz)

4. 핵심 비즈니스 로직

💵 시급 계산 공식
번 돈 = (시급 ÷ 3600) × 화장실_초

• 원탭 모드: 기본 5분(300초) 적용
• 타이머 모드: 실제 측정 시간
• 시급 미입력 시: 한국 최저시급(9,860원) 기본값
🪙 똥코인 적립 규칙
• 원탭 체크인: +10 코인
• 타이머 체크인: +10 + (분 수 × 2) 코인
• 연속 출석 보너스:
  3일: ×1.5배 / 7일: ×2배 / 30일: ×3배
• 하루 최대 3회 체크인
🎁 리워드 시스템
• 보상형 광고 1회: +50 코인 + 토스포인트
• 하루 광고 시청 제한: 3회
• 업적 달성 보너스:
  "첫 출석": +100 코인
  "일주일 연속": +500 코인
  "100회 달성": +2000 코인

5. 토스 심사 대응 (CLAUDE.md 규칙 준수)

✅ 반영 완료
  • 인트로 화면 후 로그인 (앱 시작 시 로그인 금지)
  • 공통 네비게이션바 사용 (자체 헤더 없음)
  • viewport user-scalable=no
  • 앱 이름 한글 통일 "싸월급"
  • SDK dynamic import + isSupported
  • alert/confirm/prompt 미사용 → shadcn Dialog
  • OAuth 토큰 서버에서만 처리
  • 연동 해제 콜백 처리
⚠️ 구현 시 주의
  • 공유 시 getTossShareLink() 사용
  • 외부 링크 없이 앱 내 완결
  • 광고 load → show 순서 필수
  • 기능 스킴 URL 전부 유효 페이지
  • 브랜딩(이름/색상/로고) 전체 일치
  • 첫 화면 백버튼 = 앱 종료
