🚽 체크인(출석) 방식은 어떻게?

사용자가 "지금 똥싸러 감"을 어떻게 기록할까요?

A

⏱️ 타이머 방식 (시작/종료)

"출근 중 💩" 버튼 → 타이머 시작 → 완료 버튼 → 시간 기록 + 시급 계산

✅ 정확한 시간 측정, 시급 계산에 최적 / ❌ 종료 누르는 걸 잊을 수 있음

B

✅ 원탭 출석 체크

하루에 한 번 "오늘 쌌다!" 버튼만 누르면 끝. 시간은 평균값(5분) 자동 적용

✅ 초간단, 허들 낮음 / ❌ 시간 측정 재미 없음, 개인화 약함

C

🔄 하이브리드 (추천)

기본: 원탭 출석 (간편 모드)
선택: 타이머 시작으로 정확한 시간 측정 (프로 모드)
사용자가 원하는 방식 선택 가능

✅ 간편함 + 정밀함 둘 다 / ❌ UI 분기 필요
