💰🚽 핵심 메커니즘: "돈 번다"의 계산 방식

회사에서 똥싸면 돈 번다 — 핵심 재미 요소를 어떻게 구현할까요?

A

💵 시급 기반 계산

사용자가 월급/시급을 입력하면, 화장실에 있던 시간 × 시급으로 계산

예시: "오늘 8분 × 시급 25,000원 = 3,333원 벌었다!"

✅ 직관적, 개인화된 재미 / ❌ 민감 정보(월급) 입력 필요

B

🪙 가상 재화 시스템 (똥코인)

체크인할 때마다 "똥코인" 적립 + 연속 출석 보너스 + 랭킹

예시: "3일 연속 출석! 💩×3 보너스로 +50 똥코인"

✅ 게이미피케이션, 리텐션 / ❌ 가상 재화만으로는 동기 약할 수 있음

C

🎁 실제 리워드 연동

보상형 광고 시청 → 토스포인트 실제 지급 (앱인토스 프로모션)

예시: "광고 보고 50 토스포인트 GET! 이번 달 총 1,200P"

✅ 실제 보상, 수익화 / ❌ 광고 의존, 프로모션 예산 필요

D

🔥 복합형 (추천)

시급 계산(재미) + 똥코인(게이미피케이션) + 광고 리워드(실제 보상)

예시: 체크인 → 시급 계산 보여주기 → 똥코인 적립 → 광고로 토스포인트

✅ 재미+리텐션+수익 모두 / ❌ 복잡도 증가, MVP 범위 조절 필요
